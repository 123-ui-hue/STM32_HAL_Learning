# STM32 HAL库 从零学习路线与实战记录
> 物联网工程 | 嵌入式开发 | 求职向学习仓库

## 📌 项目说明
本仓库用于记录STM32 HAL库的完整学习过程，包含**从零基础到项目实战**的代码、笔记、踩坑总结，全程基于CubeMX + HAL库开发，不依赖标准库，适合嵌入式/物联网方向求职参考。
